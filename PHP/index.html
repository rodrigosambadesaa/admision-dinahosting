Ejercicio 1

Fibonacci en PHP CLI

Este ejercicio está implementado como aplicación de línea de comandos en PHP/fibonacci.php, con clases, interfaces y separación por responsabilidades.

Además del formato real Y-m-d H:i:s, admite un modo de prueba extrema con ts:<bigint> para timestamps sintéticos gigantes. No tienen por qué corresponder a fechas físicamente posibles; sirven para comprobar el comportamiento con enteros arbitrarios.

Comandos de comprobación:

php PHP/fibonacci.php "2026-06-01 00:00:00" "2026-06-30 23:59:59"
docker compose run --rm php-cli php PHP/fibonacci.php "2026-06-01 00:00:00" "2026-06-30 23:59:59"
php PHP/fibonacci.php "ts:123456789012345678901234567890" "ts:123456789012345678901234568500"

Qué calcula:

  • Timestamps Fibonacci entre el inicio y fin del mes actual.
  • Timestamps Fibonacci entre el inicio y fin del año actual.
  • Timestamps Fibonacci entre dos fechas personalizadas en formato Y-m-d H:i:s y UTC.
  • Timestamps Fibonacci entre dos límites sintéticos enormes usando ts:<bigint>.
